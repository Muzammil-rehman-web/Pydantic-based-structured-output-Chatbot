- Uses **Groq LLaMA-3.3-70B-Versatile**

- Reliable **summary & sentiment extraction**

Using Pydantic ensures:
- Schema validation
- Predictable outputs
- Easy integration with APIs & databases
- Reduced hallucinations
- Production-grade AI pipelines
